Understand

We turn brainstorms into achieveable goals through prototyping and discussion

By taking the time to understand the business goals of each tool, we spend less time developing and more time focused on a product that will be truely useful and sustainable. Our products centralize around using data for decision-making by using solid data management, online and offline visualizations using charts, graphs, and maps, and multiple types of analysis.

Structure

Data is the core of our work. We ensure all data is readable by computers and humans making it useable for the future

We help organizations bring clarity to a variety of data types using a variety of tools. For each project we strive for open data standards where appropriate by incorporating an API [[link on what api is]] for each project.

Build

Application development is not hard, it is capturing the business process that is the challenge. We leverage and tailor open source software to specific use cases to ensure maximum utility.

By using and providing open source software we cut the development time down to fraction compared to starting from scratch. This allows for more iterations in the agile software development process [[link to what agile is]] and cuts out useless license fees.

Scale

We believe in open software coupled with open data

Through the use of APIs [[API link again]], we can rapidly prototype projects, integrate multiple data sources using open data standards, and enterprise level software to handle multiple business processes
